Parameter Description:

Existing Information:

First message (system): Defines the game rules for this TRPG session (e.g., VHS rules).

Second message (assistant): Provides the background story (script).

Generation Requirements:

Game Rules Summary:
Based on the system content from the first message, briefly restate the rule type in 1–2 sentences (e.g., “This session uses the ‘VHS’ horror role-playing rules, emphasizing immersive narrative and fear-driven decisions”), then explain the worldview in 1–2 sentences, including whether supernatural abilities exist or other key elements.

Background Story Overview:
Based on the assistant content from the second message, extract the era, core conflict, and the reason players enter the story, and summarize them in 3–4 sentences. If the player has not created a character, add a prompt at the end: “Please describe your character background (e.g., profession, special items) to start the game.” If the character is already created, briefly integrate the character into the narrative (e.g., “As investigative journalist Lin Xia, you arrive in Kyoto carrying an old camera and a protective charm”). Avoid excessive spoilers.

Game Objectives:
Clearly define the player’s core objective (e.g., “Investigate the connection between the mysterious letter and the cursed teahouse”) and survival objective (e.g., “Survive the supernatural threats in midnight Kyoto until dawn”), ensuring logical alignment with the script while avoiding spoilers. Do not explicitly reveal specific threats (e.g., if the script includes a “fox spirit,” it should instead be referred to ambiguously as “a mysterious force,” depending on context).

Key Logic:

Rule Restatement: Only emphasize rule type and core mechanics (e.g., “immersive narrative,” “choice-driven”), without detailed explanation.

Background Extraction: Focus on “why the player is here” (mysterious letter), “environmental anomalies” (empty station, supernatural phenomena), and “character motivation” (e.g., a journalist’s drive to uncover truth).

Objective Structuring: Distinguish between the main objective (investigating the truth) and the survival objective, enhancing urgency.

Interactive Guidance: If no character is created, use open-ended prompts to encourage player input (e.g., “Are you carrying special items?” “Why are you here?”). If a character exists, use a “status confirmation” approach to transition into gameplay.
